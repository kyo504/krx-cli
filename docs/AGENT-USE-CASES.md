# krx-cli AI 에이전트 활용 가이드

krx-cli를 AI 에이전트와 연동하면, 자연어 대화만으로 한국 주식시장 데이터를 조회하고 분석할 수 있습니다. CLI(Bash tool)든 MCP든 동일한 데이터와 기능을 사용할 수 있으며, 아래 시나리오는 연동 방식에 관계없이 모두 동작합니다.

설치 및 연동 방법은 [README](../README.md#ai-에이전트-연동)를 참고하세요.

> **참고**: KRX Open API 데이터는 **T-1(전 거래일) 기준 확정 데이터**이며, 실시간 시세가 아닙니다. 이 문서에서 "오늘"은 "가장 최근 거래일"을 의미합니다.

## 1. 일상 투자 정보 조회

가장 기본적인 활용입니다. 증권 앱을 열지 않고 대화만으로 시장 정보를 확인합니다.

### 시나리오

| 사용자 요청            | 에이전트가 호출하는 Tool                |
| ---------------------- | --------------------------------------- |
| "오늘 코스피 어때?"    | `krx_index` (endpoint: kospi_dd_trd)    |
| "삼성전자 주가 알려줘" | `krx_search` → `krx_stock` (isuCd 포함) |
| "오늘 금 시세 얼마야?" | `krx_commodity` (endpoint: gold_dd_trd) |
| "국채 수익률 보여줘"   | `krx_bond` (endpoint: kts_bydd_trd)     |

### 대화 예시

```
사용자: 삼성전자 주가 알려줘

에이전트: (내부적으로 krx_search → krx_stock 호출)

삼성전자(005930) 최근 거래일 정보입니다:
- 종가: 72,400원
- 전일대비: +1,200원 (+1.68%)
- 거래량: 12,345,678주
- 거래대금: 893,456백만원
- 시가: 71,500원 / 고가: 72,800원 / 저가: 71,200원
```

```
사용자: 시장 전체적으로 어때?

에이전트: (krx_market_summary 호출)

최근 거래일 시장 요약입니다:

[KOSPI] 2,645.32 (+0.87%)
- 상승: 523종목 / 하락: 312종목
- 거래대금: 8.2조원

[KOSDAQ] 892.15 (+1.23%)
- 상승: 876종목 / 하락: 456종목
- 거래대금: 6.1조원

거래대금 상위: 삼성전자, SK하이닉스, LG에너지솔루션...
상승률 상위: OO전자(+29.8%), XX바이오(+18.5%)...
```

## 2. 포트폴리오 모니터링

워치리스트에 보유 종목을 등록해두면, 한 번의 질문으로 전체 포트폴리오 현황을 파악할 수 있습니다.

### 초기 설정 대화

```
사용자: 내 포트폴리오 종목 등록해줘. 삼성전자, SK하이닉스, NAVER, 카카오, LG에너지솔루션

에이전트:
  1. krx_search/krx stock search로 각 종목 검색
  2. 복수 결과가 나오면 보통주 선택 (예: "삼성전자" → 삼성전자보통주, 삼성전자1우선주)
  3. krx_watchlist add로 등록

워치리스트에 5개 종목을 등록했습니다:
1. 삼성전자보통주 (KR7005930003)
2. 에스케이하이닉스보통주 (KR7000660001)
3. NAVER보통주 (KR7035420009)
4. 카카오보통주 (KR7035720002)
5. LG에너지솔루션보통주 (KR7373220003)
```

> **참고**: 종목명 검색 시 보통주/우선주가 모두 매칭되면 에이전트가 보통주를 선택합니다. KRX 공식 종목명에는 "보통주" 접미사가 포함되어 있습니다 (예: "삼성전자보통주"). 에이전트가 이 과정을 자동으로 처리합니다.

### 일일 모니터링

```
사용자: 내 포트폴리오 어때?

에이전트: (krx_watchlist action: show 호출)

최근 거래일 포트폴리오 현황입니다:

| 종목 | 종가 | 등락률 | 거래량 |
|------|------|--------|--------|
| 삼성전자 | 72,400 | +1.68% | 12.3M |
| SK하이닉스 | 185,000 | +2.22% | 5.6M |
| NAVER | 215,500 | -0.46% | 1.2M |
| 카카오 | 48,900 | +0.82% | 3.4M |
| LG에너지솔루션 | 378,000 | -1.05% | 0.8M |

전체적으로 5종목 중 3종목 상승, 2종목 하락입니다.
반도체 섹터(삼성전자, SK하이닉스)가 강세를 보이고 있습니다.
```

### Resource 활용

에이전트는 `krx://watchlist` Resource를 읽어 별도 Tool 호출 없이도 현재 워치리스트 상태를 파악할 수 있습니다. Rate limit 현황(`krx://rate-limit`)도 Resource로 확인 가능합니다.

## 3. 시장 분석 리포트 생성

여러 Tool을 조합하여 에이전트가 종합적인 시장 분석 리포트를 작성합니다.

### 대화 예시

```
사용자: 오늘 시장 마감 후 리포트 좀 써줘. 코스피/코스닥 동향, 섹터별 흐름,
        특이 종목 정리해서.

에이전트: (순차적으로 여러 Tool 호출)
  1. krx_market_summary → 시장 요약
  2. krx_index (kospi) → KOSPI 전 지수
  3. krx_index (kosdaq) → KOSDAQ 전 지수
  4. krx_stock (kospi, sort: FLUC_RT, limit: 10) → 상승률 Top 10
  5. krx_stock (kospi, sort: FLUC_RT, sort_direction: asc, limit: 10) → 하락률 Top 10
  6. krx_stock (kospi, sort: ACC_TRDVAL, limit: 10) → 거래대금 Top 10
  7. krx_etp (etf, sort: FLUC_RT, limit: 5) → ETF 등락률 상위

==== 2026년 3월 20일 시장 마감 리포트 ====

1. 시장 개요
   KOSPI 2,645.32 (+0.87%), KOSDAQ 892.15 (+1.23%)
   양 시장 모두 상승 마감. 상승 종목 수가 하락 종목의 1.7배...

2. 섹터별 동향 (KOSPI 시리즈 지수 기준)
   - KRX 반도체: +2.1%
   - KRX 2차전지: -0.8%
   - KRX 바이오: +1.5%...

3. 특이 종목 (상승률/하락률/거래대금 상위)
   [급등] OO전자 +29.8% (거래대금 234B)
   [급락] XX건설 -15.2% (거래대금 89B)
   [거래 폭증] YY바이오 거래대금 567B (Top 5)...

4. ETF 동향
   KODEX 반도체 +2.3%, TIGER 2차전지 -1.1%...
```

### 활용 포인트

- 에이전트가 여러 Tool을 **자율적으로 조합**하여 데이터를 수집
- 수집된 정량 데이터에 에이전트의 **분석/해석 능력**을 결합
- 매일 같은 요청을 하면 **일관된 형식**의 리포트를 받을 수 있음
- 결과를 파일로 저장하거나 노트앱에 정리하도록 추가 요청 가능

## 4. 종목 스크리닝

특정 조건에 맞는 종목을 필터링하여 투자 후보군을 탐색합니다.

### 대화 예시

```
사용자: 코스피에서 오늘 5% 이상 오른 종목 찾아줘

에이전트: (krx_stock 호출, filter: "FLUC_RT > 5", sort: FLUC_RT)

오늘 KOSPI 시장에서 5% 이상 상승한 종목입니다:

| 종목명 | 종가 | 등락률 | 거래대금 |
|--------|------|--------|----------|
| OO전자 | 15,200 | +29.8% | 234,567M |
| XX소재 | 8,900 | +12.5% | 45,678M |
| YY화학 | 32,100 | +7.8% | 123,456M |
| ... | ... | ... | ... |

총 18종목이 해당됩니다.
```

```
사용자: 거래대금 상위 20개 종목 중에서 하락한 종목만 보여줘

에이전트:
  1. krx_stock (kospi, sort: ACC_TRDVAL, limit: 20) → 거래대금 Top 20
  2. 결과에서 FLUC_RT < 0 인 종목 필터링

거래대금 Top 20 중 하락 종목입니다:

| 종목명 | 종가 | 등락률 | 거래대금 |
|--------|------|--------|----------|
| AA건설 | 45,600 | -3.2% | 567,890M |
| BB엔터 | 12,300 | -1.5% | 345,678M |

2종목이 해당됩니다.
거래대금이 높은데 하락한 종목은 매도 압력이 강한 것으로 볼 수 있습니다.
```

```
사용자: ETF 중에서 레버리지/인버스 빼고 거래량 많은 순으로 10개만

에이전트: (krx_etp 호출, sort: ACC_TRDVOL, limit: 10,
          filter로 레버리지/인버스 제외)

일반 ETF 거래량 Top 10입니다:

1. KODEX 200 - 거래량 5,234,567주
2. TIGER 반도체 - 거래량 3,456,789주
...
```

## 5. 기간별 추세 분석

`date_from`/`date_to` (CLI에서는 `--from`/`--to`) 파라미터로 여러 날짜의 데이터를 한 번에 조회하여 추세를 분석합니다. 날짜별로 병렬 호출하므로 한 번의 요청으로 수십 거래일의 데이터를 받을 수 있습니다.

### 대화 예시

```
사용자: 삼성전자 최근 2주간 주가 추이 알려줘

에이전트: (krx_stock 호출, date_from: 20260306, date_to: 20260320,
          isuCd: KR7005930003)

삼성전자(005930) 최근 2주 주가 추이입니다:

| 날짜 | 종가 | 등락률 | 거래량 |
|------|------|--------|--------|
| 03/06 | 69,800 | -0.57% | 11.2M |
| 03/09 | 70,200 | +0.57% | 10.8M |
| 03/10 | 71,000 | +1.14% | 13.5M |
| ... | ... | ... | ... |
| 03/20 | 72,400 | +1.68% | 12.3M |

2주간 +3.7% 상승했습니다.
3/10 이후 꾸준히 상승세를 유지하고 있으며,
거래량도 점진적으로 증가하는 패턴입니다.
```

```
사용자: 이번 달 코스피 지수 흐름을 차트로 설명해줘

에이전트: (krx_index 호출, market: kospi, date_from: 20260301, date_to: 20260320)

3월 KOSPI 지수 흐름입니다:

2,620 |                                    *
2,610 |                              * *  *
2,600 |          *  *           *  *
2,590 |    *  *     *  *     *
2,580 | *              *  *
      +----+----+----+----+----+----+----
       3/3  3/5  3/7 3/10 3/12 3/14 3/17 3/19

- 월초 2,580대에서 출발하여 현재 2,645까지 상승 (+2.5%)
- 3/7~3/12 조정 구간 이후 반등
- 최근 5거래일 연속 상승 중
```

## 6. 멀티 자산 비교 분석

주식, ETF, 채권, 원자재 등 서로 다른 자산군의 데이터를 조합하여 비교합니다.

### 대화 예시

```
사용자: 오늘 주식, 채권, 금, 유가 전체적으로 어떻게 됐는지 한눈에 정리해줘

에이전트: (병렬로 여러 Tool 호출)
  1. krx_index (kospi) → 주식
  2. krx_bond (kts) → 국채
  3. krx_commodity (gold) → 금
  4. krx_commodity (oil) → 유가

오늘의 멀티 자산 현황입니다:

| 자산군 | 지표 | 현재 | 등락 |
|--------|------|------|------|
| 주식 | KOSPI | 2,645.32 | +0.87% |
| 주식 | KOSDAQ | 892.15 | +1.23% |
| 채권 | 국고채 3년 | 3.125% | -0.02%p |
| 채권 | 국고채 10년 | 3.456% | -0.03%p |
| 원자재 | KRX 금 (g) | 92,450원 | +0.5% |
| 원자재 | KRX 원유 | 78,200원 | -1.2% |

분석:
- 주식/채권 동반 상승 (금리 하락 → 채권 가격 상승)
- 금 소폭 상승, 유가 하락 → 인플레이션 압력 완화 시그널
- Risk-on 분위기로 해석됩니다
```

```
사용자: KODEX 200 ETF와 KOSPI 지수의 괴리율 확인해줘

에이전트:
  1. krx_index (kospi, fields: [CLSPRC_IDX]) → KOSPI 종가
  2. krx_etp (etf, isuCd: KODEX200코드, fields: [TDD_CLSPRC, NAV]) → ETF 종가/NAV

KODEX 200 vs KOSPI 200 괴리율:
- KODEX 200 시장가: 35,425원
- NAV(순자산가치): 35,410원
- 괴리율: +0.04% (거의 없음)
- 정상 범위 내에서 추적이 잘 되고 있습니다.
```

## 7. 리스크 모니터링

급변하는 시장 상황이나 특정 이벤트 발생 시 빠르게 상황을 파악합니다.

### 대화 예시

```
사용자: 오늘 하한가 종목 있어?

에이전트: (krx_stock, filter: "FLUC_RT < -25", sort: FLUC_RT, sort_direction: asc)

오늘 코스피+코스닥에서 -25% 이상 하락한 종목입니다:

| 종목 | 시장 | 종가 | 등락률 | 사유 |
|------|------|------|--------|------|
| XX제약 | KOSDAQ | 3,200 | -29.9% | 하한가 |
| YY홀딩스 | KOSPI | 8,100 | -26.3% | - |

2종목이 급락했습니다. 공시 확인을 권장드립니다.
```

```
사용자: 내 워치리스트 종목 중에 오늘 3% 이상 빠진 거 있어?

에이전트:
  1. krx://watchlist Resource로 워치리스트 확인
  2. krx_watchlist (action: show) 호출
  3. 등락률 < -3% 필터링

워치리스트 종목 중 오늘 3% 이상 하락한 종목:

- LG에너지솔루션: -3.8% (378,000원)
  → 최근 5일 -7.2% 하락 추세

나머지 4종목은 -3% 이내입니다.
주의가 필요합니다.
```

```
사용자: 파생상품 시장 변동성은 어때?

에이전트:
  1. krx_derivative (opt_bydd_trd) → 옵션 전체 데이터 (12,000건+)
  2. RGHT_TP_NM(CALL/PUT) 기준으로 ACC_TRDVOL 합산하여 P/C Ratio 계산
  3. krx_derivative (fut_bydd_trd) → 선물 데이터

KOSPI200 옵션 시장 현황:
- CALL 총거래량: 852,457계약
- PUT 총거래량: 735,870계약
- Put/Call Ratio: 0.86

선물 시장:
- KOSPI200 선물: 351.25 (-0.92%)
- 미결제약정(ACC_OPNINT_QTY): 245,678계약

## 11. 전략 백테스트

KRX Open API는 2010년부터의 일별 OHLCV(시가/고가/저가/종가/거래량) 데이터를 제공합니다. `date_from`/`date_to`로 과거 데이터를 가져와 에이전트가 직접 전략을 검증하거나, 백테스트 코드를 생성할 수 있습니다.

### 사용 가능한 데이터

| 자산 | 필드                                                                      | 비고                    |
| ---- | ------------------------------------------------------------------------- | ----------------------- |
| 주식 | TDD_OPNPRC, TDD_HGPRC, TDD_LWPRC, TDD_CLSPRC, ACC_TRDVOL, FLUC_RT, MKTCAP | 종목별 isuCd 필터       |
| 지수 | OPNPRC_IDX, HGPRC_IDX, LWPRC_IDX, CLSPRC_IDX, ACC_TRDVOL, FLUC_RT         | IDX_NM으로 지수 선택    |
| ETF  | TDD_CLSPRC, NAV, ACC_TRDVOL, IDX_IND_NM, FLUC_RT                          | NAV로 괴리율 추적       |
| 선물 | TDD_CLSPRC, SETL_PRC, ACC_OPNINT_QTY                                      | 정산가, 미결제약정 포함 |

### 시나리오 A: 대화 내에서 간단한 백테스트

```

## 주의사항

- **데이터 시차**: KRX 데이터는 T-1(전 거래일) 기준 확정 데이터이며, 실시간 시세가 아닙니다
- **Rate Limit**: 일 10,000건 제한이 있으므로, 기간 조회 시 날짜 범위를 적절히 설정하세요
- **서비스 승인**: 각 카테고리(주식, 채권, 파생상품 등)별로 KRX Open API 포털에서 별도 승인이 필요합니다
- **제공하지 않는 데이터**: KRX Open API는 다음 데이터를 포함하지 않습니다. 필요 시 DART, 증권사 API 등을 병행하세요:
  - 배당 정보 (배당금, 배당수익률)
  - 투자자별 매매 동향 (외국인/기관/개인 순매수)
  - 재무제표 (매출, 영업이익, PER/PBR 등)
  - 공시/뉴스
  - 실시간 호가/체결
- **에이전트 해석**: 에이전트가 데이터에 덧붙이는 분석(섹터 전망, 시장 해석 등)은 에이전트 자체 지식에 기반한 것이며, KRX API에서 제공하는 정보가 아닙니다
- **투자 판단**: 에이전트의 분석은 참고용이며, 최종 투자 판단은 본인 책임입니다
